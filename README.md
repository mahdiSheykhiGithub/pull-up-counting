# شمارش حرکت پول آپ (Pull-Up Counter)
این پروژه به منظور شمارش تعداد حرکات ورزشی پول آپ انجام شده در یک ویدیو با استفاده از مدل تخمین ژست YOLOv8 طراحی شده است. در این پروژه از یک اسکریپت پایتون استفاده می‌شود که ویدیو را فریم به فریم پردازش می‌کند. این اسکریپت به شرح زیر عمل می‌کند:

وارد کردن ویدیو: ابتدا، اسکریپت ویدیو را از مسیر مشخص شده می‌خواند.

تشخیص نقاط کلیدی با استفاده از YOLOv8: مدل YOLOv8 برای تشخیص نقاط کلیدی (Keypoints) بدن انسان در هر فریم از ویدیو استفاده می‌شود. این نقاط کلیدی شامل محل‌های خاصی از بدن مانند شانه‌ها، آرنج‌ها، و مچ‌ها هستند.

شناسایی حرکات پول آپ: با استفاده از نقاط کلیدی شناسایی شده، اسکریپت موقعیت شانه‌ها و آرنج‌ها را در هر فریم بررسی می‌کند. برای شناسایی یک حرکت پول آپ، اسکریپت بررسی می‌کند که آیا آرنج‌ها به بالای شانه‌ها حرکت کرده‌اند یا خیر. اگر آرنج‌ها به بالا حرکت کنند و سپس به پایین برگردند، این به عنوان یک حرکت پول آپ کامل شمارش می‌شود.

شمارش حرکات: هر بار که یک حرکت کامل پول آپ شناسایی می‌شود، شمارنده افزایش می‌یابد. اسکریپت به طور مداوم فریم‌ها را پردازش می‌کند و تعداد حرکات پول آپ را به‌روزرسانی می‌کند.

نمایش نتایج: تعداد حرکات پول آپ شناسایی شده به صورت زنده بر روی ویدیو نمایش داده می‌شود. این نتایج به کاربر نشان می‌دهد که چند حرکت پول آپ به درستی انجام شده است.

به طور کلی، این پروژه یک ابزار خودکار برای شمارش حرکات ورزشی پول آپ است که می‌تواند در تحلیل عملکرد ورزشی و بهبود تمرینات بدنی مفید باشد.

. **پردازش هر فریم برای شناسایی نقاط کلیدی و شمارش پول آپ‌ها:**

    ```python
    while cap.isOpened():
        ans, frame = cap.read()
        key = cv2.waitKey(1)
        if key == ord('q'):
            break

        results = model.predict(frame, show=False)
        
        pointL1 = results[0].keypoints.xy.numpy()[0][5].astype(int)
        pointL2 = results[0].keypoints.xy.numpy()[0][7].astype(int)

        pointR1 = results[0].keypoints.xy.numpy()[0][6].astype(int)
        pointR2 = results[0].keypoints.xy.numpy()[0][8].astype(int)

        try:
            if (not is_count) and (pointL1[1] < pointL2[1]) and (pointR1[1] < pointR2[1]):
                counter += 1
                is_count = True
            elif (pointL1[1] > pointL2[1]) and (pointR1[1] > pointR2[1]):
                is_count = False
        except:
            pass

        frame = cv2.putText(frame, f'pull-up:{counter}', (450, 320), cv2.FONT_HERSHEY_COMPLEX, 1, (255, 255, 255), 3)
        out.write(frame)

    cap.release()
    out.release()
    cv2.destroyAllWindows()
    ```

## آزمایش و ارزیابی مدل با ویدیوهای نمونه
دو ویدیوی کوتاه برای آزمودن مدل به سیستم داده شدند. این ویدیوها به منظور ارزیابی دقت و کارایی مدل در شناسایی و شمارش حرکات پول آپ استفاده شدند. نتایج پردازش این ویدیوها در قالب دو فایل با نام‌های pull_up_counting1 و pull_up_counting2 ذخیره شده‌اند. این فایل‌ها برای دانلود و بررسی‌های بیشتر در دسترس هستند. با مشاهده این ویدیوها می‌توانید عملکرد مدل را در تشخیص حرکات پول آپ در شرایط مختلف بررسی کنید و با دقت و کارایی آن آشنا شوید.
